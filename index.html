# What is this ?

This platform Provides users with the ability to voice chat with others in real-time. Users can choose to use any mode of communication to chat with others.

# How to Start ?

If you're reading this, you've probably already filled your NAME and CHANNEL ID. If not, then Close the Current Tab and reopen this website. Then, You'll get Prompt asking for your NAME and CHANNEL ID.

# What is Channel ID ?

Channel ID is ID of a virtual room, where people can join the conversation with others. Each room can fit multiple users and allows them to Chat with each other.
When Opening this WebPage, You were assigned a random Channel, Which You can Change to whatever you want.
Users with you Channel ID will be able to join your channel. So, you may want to share your Channel ID with your Friends and Family.

# Then What ?

After Getting your Name and Channel Id, Click on Join Chat button to enter the Chatroom. And Now, You can see Other Users in the currenr Channel.

# How to Chat ?

Once You've Joined The Room, You can use either Voice Chat or Texting or both to communicate with other Users.

All Good, but Privacy ?
